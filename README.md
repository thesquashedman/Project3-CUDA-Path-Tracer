### Introduction
This is a CUDA based ray tracer. It supports loading of GLTF meshes and their diffuse textures. It supports two types of materials, a diffuse material and a perfectly specular material. It is optimized with stream compaction to eliminate finished rays, and a BVH spatial structure to speed up triangle ray intersections.
